## dependencies
node
npm

## installation
* create folder "spirals" somewhere
* open terminal
 ```
 npm init 
 npm i @riketscience/spirals
```
* open editor (e.g. VSCode) in spirals folder
* within spirals directory add spiral.js :

	  const spiral = require("@riketscience/spirals");
	  spiral();

back in terminal
> node spiral 18
